## 📌 Project Overview

This project analyzes **9,800 sales records** to uncover business insights including revenue trends, regional performance, and product-level analysis.

The goal was to simulate a real-world data analytics workflow:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Time-Series Analysis
- Business Insight Extraction

---

## 📂 Dataset Information

- Records: 9,800
- Features: 18 columns
- Key Columns:
  - Order Date
  - Ship Date
  - Region
  - Category
  - Product Name
  - Sales
